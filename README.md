## Getting Started

- Flutter SDK >= 3.2.6

### Installation

```bash

git clone https://github.com/YOUR_USERNAME/nz_trip.git

# Navigate to project directory
cd nz_trip

# Install dependencies
flutter pub get

# Run the app
flutter run
```
